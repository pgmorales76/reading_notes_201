## *What does a CSS transform allow the developer to do to an element?*

### It allows a developer to change the shape and position of the affected content without disrupting the normal document flow

## *Provide an example of a transform and how you could see that being used on a website.*

### If you wanted to rotate an image, translate (affect its location in the browser window) an image, or scale (make larger, or smaller) an image

[CSS Transforms](http://learn.shayhowe.com/advanced-html-css/css-transforms/)

## *What does a CSS transition allow the developer to do to an element?*

### Transitions enable you to define the transition between two states of an element. Different states may be defined using pseudo-classes like `:hover`, `:active`, or dynamically set using JavaScript

## *How does a CSS animation differ from a CSS transition?*

### CSS animations are used when more control is required, when transitions need to have multiple states

[CSS Transitions & Animations](http://learn.shayhowe.com/advanced-html-css/transitions-animations/)

## *What are some benefits to using CSS transitions on websites?*

### Transitions will excite your users, increase engagement and ultimately, when used well, increase your conversions. Also, these effects are hardware accelerated, and a progressive enhancement that can be used, now
